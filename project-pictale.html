PicTale

Django-based Full-Stack Web Application

Duration: June 2025 Full-Stack Development
GitHub Live Demo

Overview

PicTale is a Django-based full-stack web application that supports full CRUD operations for photos and videos. It includes secure user authentication, media upload, edit/delete functionality, and an admin dashboard.

Problem

Need for a centralized platform to manage, organize, and share photos and videos with secure user authentication and full control over media content.

Solution

Built a complete web application with Django's robust backend, implementing CRUD operations, user authentication system, and responsive dark UI for seamless media management.

Key Features

▹

User Authentication System

Secure login and registration with session management

▹

Media Upload

Support for photos and videos with efficient storage

▹

CRUD Operations

Complete create, read, update, delete functionality

▹

Responsive Dark UI

Modern, clean interface optimized for all devices

Tech Stack

Django Python Tailwind CSS Bootstrap HTML ORM
